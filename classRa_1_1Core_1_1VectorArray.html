Detailed Description

template<typename V>
class Ra::Core::VectorArray< V >

This class is a wrapper around a std::vector of Core::Vectors.

which allow to use the stdlib's dynamic array implementation, yet pass it as a matrix when Eigen needs it with the getMap() method.
