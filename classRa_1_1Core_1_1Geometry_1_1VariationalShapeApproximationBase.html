Detailed Description

template<uint K_Region>
class Ra::Core::Geometry::VariationalShapeApproximationBase< K_Region >

The VariationalShapeApproximationBase class.

The VariationalShapeApproximationBase class computes the K proxies describing a surface from a given triangle mesh.

Note
It is based on "Variational Shape Approximation" paper.
Warning
It doesn't implement region teleporting or non-organic shape partitioning.
