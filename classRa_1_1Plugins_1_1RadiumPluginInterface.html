virtual void Ra::Plugins::RadiumPluginInterface::registerPlugin ( const PluginContext &  context)
pure virtual

Pass arguments for plugin initialization.

This method is supposed to create the system (<emph>new</emph> it) and then call Ra::Engine::RadiumEngine::registerSystem(). Without doing this, your system won't do anything (unless you just want to add UI elements)

Parameters
context: plugin context containing the engine and UI interfaces.
